# Pubgm-viewership-analysis
This project analyzes PUBG Mobile (PUBGM) esports viewership data to understand audience engagement patterns across different tournaments, regions, and streaming platforms. The analysis focuses on identifying peak viewership periods, popular events, and platform-wise distribution to uncover trends that drive esports popularity.

Using Python, Pandas, and data visualization techniques, the project involves data cleaning, exploratory data analysis (EDA), and insight generation. The objective is to demonstrate practical esports/sports analytics skills and present data-driven insights that can help tournament organizers, teams, and sponsors make informed decisions.
